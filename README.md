# Pre-requisites

## Option 1: VS Code Web (Bash Shell), Link to Open : https://vscode.dev/azure

### Verify Python Installation

```bash
python3 --version
```

### Create and Activate Virtual Environment

#### Step A: Open Project Folder

- Locate or create your working folder.
- Right-click the folder.
- Select **Open in Integrated Terminal**.

#### Step B: Create Virtual Environment

```bash
python3 -m venv myenv
```

#### Step C: Activate Virtual Environment

```bash
source myenv/bin/activate
```

#### Step D: Install Required Packages

```bash
pip install -r requirements.txt
```

Or

```bash
pip install --user -r requirements.txt
```

---

## Option 2: VS Code Desktop (PowerShell)

### Verify Python Installation

```powershell
python --version
```

### Create and Activate Virtual Environment

#### Step A: Open Project Folder

- Locate or create your working folder.
- Right-click the folder.
- Select **Open in Integrated Terminal**.

#### Step B: Create Virtual Environment

```powershell
python -m venv labenv
```

#### Step C: Activate Virtual Environment

```powershell
.\labenv\Scripts\Activate
```

#### Step D: Install Required Packages

```powershell
pip install -r requirements.txt
```

After activating the virtual environment, confirm that Python is available:
